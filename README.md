去重包括：

    1. 原视频每十分之一的段落会被随机调速
    
    2. 每十分之一的段落会被删除1帧
    
    3. 视频轻度锐化
    
    3. 增加滤镜 (亮度、伽马、对比度等)
    
    4. 支持视频镜像
    
    5. 支持 放大/横屏转竖屏(模糊背景)/竖屏转横屏(模糊背景) 三种模式
    
    note: 视频处理不会影响音频，音频在视频处理前会抽离，处理后又加入进去
    

Program.cs 里可开启自动更新 和 授权码验证 （不用就不用管）
